# Navigation Images

This folder contains the navigation icons for the website header.

## Current Placeholder Images

- `nav-work.png` - 24x24px (Work tab)
- `nav-about.png` - 44x24px (About tab) 
- `nav-case-studies.png` - 24x24px (Case studies tab)

## Replacing Images

To replace these placeholder images with your own:

1. **Keep the exact filenames**: `nav-work.png`, `nav-about.png`, `nav-case-studies.png`
2. **Maintain the exact dimensions**:
   - Work: 24x24px
   - About: 44x24px  
   - Case studies: 24x24px
3. **Use PNG format** for best quality
4. **Optimize file size** for web performance

## Design Guidelines

- Images should be clear and recognizable at small sizes
- Use consistent visual style across all navigation icons
- Consider both light and dark theme compatibility
- Ensure good contrast with the navigation background

The images will automatically appear in the navigation header once you replace the placeholder files.
